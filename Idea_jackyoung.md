# 나만의 칵테일 냉장고

   
   
   
### 서비스 개요
1. 내가 가지고 있는 술 종류, 재료 저장하면
2. 만들 수 있는 칵테일을 추천해준다.
3. 추가로 사야할 재료들을 추천해준다.
4. 나만의 레시피나 관심있는 술 종류도 추천해 준다.(그 과정에서 데이터 사이언스나 머신러닝 기법 사용 가능)
   
   
   
### 핵심 기능
+ 나만의 술 냉장고 기능
+ 재료 추천 기능
+ 칵테일 추천 기능
+ 칵테일 만드는 **동영상** 기능
+ 사용자가 직관적으로 알 수 있는 인터페이스
